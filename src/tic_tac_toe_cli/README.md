# tic-tac-toe-clj

## How to Play

### Locally with another player

* `lein repl` inside the root directory
* Inside the repl `(use '[tic-tac-toe-clj.cli])`
* `(play-cli)`

### Locally against unbeatable ai

* `lein repl` inside the root directory
* Inside the repl `(use '[tic-tac-toe-clj.cli])`
* `(play-cli :ai)`
  `
